Comment ça marche

  1. Tu récupères manuellement la dernière IPA Instagram décryptée sur decrypt.day (bouton Download for Free).
  2. Tu la dépose dans la zone d'upload ci-dessus. L'upload streame directement sur le serveur (~280 Mo).
  3. Optionnel : sélectionne un patch à appliquer après l'injection SCInsta.
  4. Clique Builder maintenant. Un conteneur Docker : clone SoCuul/SCInsta main (frais), lance ./build.sh sideload (Theos + cyan + ipapatch), applique le patch choisi, dépose l'IPA final dans le store.
  5. Le serveur crée automatiquement une nouvelle Version pour com.burbn.instagram et publie un article de news.
